Visualization for Palindrome Example (121)

reverseNum (121, 0)
= reverseNum(12, 1)        // rev = 0*10+1=1
= reverseNum(1, 12)         // rev = 1*10 + 2 = 12
= reverseNum(0, 121)        // rev = 12*10+1 = 121
= 121                       // (base case reached)

So the reversed number = 121.

Finally, since 121 == 121, it's a palindrome
